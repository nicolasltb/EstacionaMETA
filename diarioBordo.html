  • 10 de Julho de 2019

    Nos encontramos com a professora/orientadora Sílvia para pedir sugestões de idéias para serem apresentadas na META.

  • 13 de Julho de 2019

    Fizemos uma reunião entre o grupo e decidimos com a ajuda da orientadora que o projeto da META apresentado será sobre gestão de estacionamentos automatizada.

  • 15 de Julho de 2019

    Nos reunimos na casa de um dos integrantes (Nicolas) para que pudéssemos fazer alguns avanços no projeto e ter uma colaboração entro o grupo para uma maior desempenho no trabalho.

  • 16 de Julho de 2019

    Foi criado um grupo no WhatsApp com a professora Sílvia para que fossem compartilhados os avanços e resultados do nosso projeto

  • 23 de Julho de de 2019

    Nos encontramos com a orientadora Sílvia e foi decidido que iríamos fazer um aplicativo usando React Native para implementar o nosso projeto.

  • 2 de Agosto de 2019

    Marcamos entre os integrantes do grupo para ir ao centro de Belo Horizonte para procurar alguns componentes que iríamos necessitar na montagem do nosso protótipo. Foram comprados para o nosso protótipo alguns sensores ultrassônicos, jumpers e um sensor RFID.

  • 11 de Agosto de 2019

    Recebemos um e-mail dizendo que o projeto foi oficialmente aprovado pela comissão organizadora da meta.

  • 15 de Agosto de 2019

    Começaram os estudos de React Native para a criação do aplicativo, porém percebemos que iríamos ter muitos problemas para utilizá-lo junto com o banco de dados e decidimos reavaliar a nossa escolha.

  • 20 de Agosto de 2019

    Nos reunimos no laboratório do CEFET para podermos testar os componentes e os códigos que foram feitos, porém muitos erros ocorreram e tivemos algumas dificuldades para corrigi-los. No final do dia tivemos resultados satisfatórios e consequentemente conseguimos corrigir alguns dos erros iniciais.

  • 14 de Setembro de 2019

    Após uma reunião entre os integrantes foi decidido a criação de uma aplicação Web para o nosso projeto, para que os dados recolhidos fossem melhor apresentados para o usuário final.

  • 20 de Setembro de 2019

    Após alguns testes e discussões entre o grupo conseguimos criar um site e que agradava os integrantes em questão de praticidade e design.

  • 30 de Setembro de 2019

    Após algumas tentativas conseguimos fazer com que a aplicação Web fosse compatível aos dispositivos móveis e com isso ter uma maior acessibilidade ao público.

  • 07 de Outubro de 2019

    Depois de vários problemas solucionados conseguimos ter um resultado satisfatório na criação do banco de dados para o controle de acesso e o gerenciamento de vagas do nosso projeto.

  • 10 de Outubro de 2019

    Tivemos diversos problemas para a conexão do protótipo com o banco de dados e consequentemente causou um atraso no projeto atrapalhando o resultado desejado do dia.

  • 12 de Outubro de 2019

    Após algumas pesquisas e consultas com alguns amigos conseguimos obter bons resultados na conexão do banco de dados e com isso tivemos um excelente progresso no projeto adiantando várias metas que tínhamos planejado.

  • 15 de Outubro de 2019

    Começamos a providenciar a maquete do protótipo, sendo decidido a utilização de uma madeira como base do estacionamento.

  • 16 de Outubro de 2019

    Participação de todos os integrantes na palestra de apresentação da META para explicações e conselhos dos organizadores do evento.

    Confeccionamos a primeira versão banner do nosso projeto. Enviamos para a revisão da nossa orientadora e chegamos a uma versão final.

  • 17 de Outubro de 2019

    Enviamos o banner para a gráfica para ser imprimido.

  • 18 de Outubro de 2019

    Ajustes realizados para a finalização dos códigos e do protótipo para a implementação na maquete e consequentemente para a apresentação da META.

  • 21 de Outubro de 2019

    Implementação do protótipo feito na maquete e a realização de alguns ajustes para o melhor desempenho possível do projeto realizado.

    Participação dos integrantes na palestra de abertura oficial da META.

  • 22 de Outubro de 2019

    Credenciamento dos integrantes do grupo e o foco na organização do nosso estande, após isso fizemos pequenos ajustes no projeto para que tudo ocorra como planejado nas apresentações seguintes.
